# Dhruv Toprani Portfolio

Premium personal portfolio for Dhruv Kekin Toprani. Built to position Dhruv for PM, TPM, AI product, robotics, and software leadership opportunities.

## Stack

- React
- Vite
- Three.js via React Three Fiber
- Framer Motion
- Lucide React

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Design direction

Institutional black, warm ivory, restrained gold, sharp typography, memo-style case studies, and subtle premium motion.
